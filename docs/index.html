Why PathZep?

📋

Right-click to copy

Right-click any file or folder in Finder to instantly copy its absolute or home-relative (~/) path.

⌨️

Global shortcuts

Set custom keyboard shortcuts to copy paths from Finder — works from any app, no clicking required.

🪶

Lightweight & native

Sits quietly in your menu bar. Built as a native macOS app with minimal resource usage.

Get started in 30 seconds

1

Download and drag to Applications

Open the DMG and drag PathZep into your Applications folder.

2

Enable the Finder extension

System Settings → Privacy & Security → Extensions → PathZep

3

Right-click or use your shortcut

The path is copied to your clipboard, ready to paste.
